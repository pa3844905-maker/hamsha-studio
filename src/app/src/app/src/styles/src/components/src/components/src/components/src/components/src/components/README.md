# Hamsha Studio — Starter Scaffold

This repo is a premium digital agency starter scaffold built with Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion, GSAP, Lucide Icons, React Hook Form, and EmailJS.

Getting started:
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Open `http://localhost:3000`

What’s included:
- Design tokens matching the requested color system and typography.
- App Router layout, global styles, and core components (Navbar, Hero, Button, Footer).
- Tailwind CSS config and PostCSS.
- Framer Motion animations for hero and buttons.
